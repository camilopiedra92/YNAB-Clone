# Tone Directions Catalogue

Choose **one** dominant direction. You may blend at most **two** — more than that produces aesthetic incoherence.

## Directions

| Direction                    | Feel                                     | Best For                                  |
| ---------------------------- | ---------------------------------------- | ----------------------------------------- |
| **Brutalist / Raw**          | Unpolished, confrontational, honest      | Developer tools, manifestos, portfolios   |
| **Editorial / Magazine**     | Typographic hierarchy, storytelling      | Content platforms, blogs, landing pages   |
| **Luxury / Refined**         | Restrained, expensive, precise           | Finance, SaaS, premium products           |
| **Retro-Futuristic**         | Nostalgic tech, CRT glow, analog+digital | Creative tools, music, entertainment      |
| **Industrial / Utilitarian** | Functional, dense, monospace             | Dashboards, admin panels, data tools      |
| **Organic / Natural**        | Soft, earth-toned, flowing shapes        | Wellness, sustainability, community       |
| **Playful / Toy-like**       | Rounded, colorful, bouncy motion         | Consumer apps, education, onboarding      |
| **Maximalist / Chaotic**     | Dense, layered, overwhelming intent      | Art, fashion, experimental platforms      |
| **Minimalist / Severe**      | Extreme reduction, tension in void       | Luxury brands, portfolios, single-purpose |

## Blending Rules

- ✅ Editorial + Luxury → refined storytelling
- ✅ Industrial + Brutalist → raw functional density
- ✅ Retro-Futuristic + Playful → nostalgic warmth
- ❌ Maximalist + Minimalist → contradiction
- ❌ Playful + Brutalist → tonal clash
- ❌ Three or more directions → loss of identity
